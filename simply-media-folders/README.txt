== Description ==

Simply adds folders to the WordPress media Library.
Accessable in the media library menu as well as media modal popups.

=== Page Builder Integration's ===

* Elementor Website Builder
* Beaver Page Builder

== Installation ==

1. Upload the `simply-media-folders` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Deactivation =

When deactivated no data is deleted. So after reactivation all folders and
associations restored automatically.

= Uninstall =

When the plugin is uninstalled all folder data is deleted.
